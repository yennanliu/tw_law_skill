# 在 ChatGPT 使用本前置提示

有兩種方式，擇一即可。

## 方式 A：自訂指令（最快，適用一般帳號）

1. 開啟 ChatGPT，點右上角頭像 →「自訂 ChatGPT（Customize ChatGPT）」。
2. 在「ChatGPT 應如何回覆？（How would you like ChatGPT to respond?）」欄位，
   貼上 `系統提示_台灣法律助手.md` 中「───」之間的全部文字。
3. 儲存。之後所有對話都會套用此設定。

## 方式 B：建立專屬 GPT（適用 Plus / Team / Enterprise）

1. 左側選單 →「探索 GPT（Explore GPTs）」→ 右上「建立（Create）」。
2. 切到「設定（Configure）」分頁：
   - **名稱**：台灣法律案件分析助手
   - **描述**：依公開裁判進行台灣法律案件之彙整、風險評估與策略建議。
   - **指示（Instructions）**：貼上 `系統提示_台灣法律助手.md` 的全部提示內容。
3. （選用）建議對話開場白：
   - 「幫我盤點某當事人的公開判決歷程」
   - 「租約到期租客不搬，我可以主張什麼？」
   - 「這份判決的請求權基礎與風險是什麼？」
4. 儲存，選擇「只有我（Only me）」以保護隱私。

## 小提醒

- 貼提示時**不要**連同 Markdown 的標題或說明一起貼，只貼「───」之間的正文。
- 不要在 GPT 的公開設定或知識檔案中放入任何真實個資。
