# 台灣法律小幫手 🧑‍⚖️

把 ChatGPT 或 Gemini 變成一位**專業的台灣法律顧問**，
幫你判斷「**這件事在台灣合不合法？**」，並用白話說明法律依據與該怎麼做。

專長：**民法、刑法、租賃住宅市場發展及管理條例**（租屋、買賣、借貸、
損害賠償、是不是犯罪⋯⋯）。

---

## 🚀 三步驟安裝（免技術、長輩也會）

只要「複製 → 貼上 → 開始問」三步，不用寫程式。

### 第 1 步：複製指令

打開這個檔案，把裡面**兩條虛線（───）中間**的全部文字，整段複製起來：

👉 **[點我打開指令檔](doc/系統提示_台灣法律顧問.md)**

> 小技巧：用手指或滑鼠從第一行虛線下方，拖到最後一行虛線上方，全部選起來再複製。

### 第 2 步：貼到 ChatGPT 或 Gemini

**用 ChatGPT：**
1. 打開 ChatGPT，點右上角**你的頭像**。
2. 點「**自訂 ChatGPT**（Customize ChatGPT）」。
3. 把剛剛複製的文字，**貼到大格子裡**，按**儲存**。

**用 Gemini：**
1. 打開 Gemini，左邊點「**Gem 管理員**」→「**新增 Gem**」。
2. 名稱打「台灣法律小幫手」，把複製的文字**貼到「指示」格子**。
3. 按**儲存**。

### 第 3 步：開始問問題

直接用自己的話問就好，例如：

> - 房東要漲租金，我可以不同意嗎？
> - 租約到期房客不搬走，我可以怎麼辦？
> - 押金可以扣到幾個月？房東不還押金怎麼辦？
> - 朋友借錢不還，可以告他嗎？
> - 有人在網路上罵我，這樣算不算犯罪？

小幫手會用**紅黃綠燈**告訴你合不合法，並說明依據和建議做法。

---

## 💡 使用小提醒

- 問的時候盡量講清楚**發生什麼事、你是哪一方**（房東還是房客？借錢還是被借？）。
- 它給的是**參考資訊，不是正式法律意見**；重要的事情，記得找**律師**或
  免費的**法律扶助基金會（電話 412-8518）**確認。
- 法律會修改，牽涉到金額或條文時，以**最新法規**為準。

## 🔒 隱私與安全

- 你打進去的個人資料（姓名、身分證、地址）只用在當下這次對話，
  請**不要**把敏感資料截圖公開分享。
- 本專案**不含任何真實個人資料**，所有範例都是化名。
