# 台灣法律案件分析助手（ChatGPT / Gemini 前置提示）

這是一個「前置提示（pre-prompt / system prompt）」技能包，用來把 ChatGPT、Gemini
等大型語言模型（LLM）調校成更懂**中華民國（台灣）法律案件分析**的助手。

把本包的系統提示貼到模型的「自訂指令 / Custom Instructions」「GPT 指示」或
「Gem 指示」欄位後，模型在協助你分析台灣裁判、盤點訴訟風險、擬定法律策略時，
會遵循一致的方法論、正確的引用格式與必要的免責與隱私原則。

## 內容

- `prompts/系統提示_台灣法律助手.md` — 核心前置提示（貼到模型設定用）
- `prompts/chatgpt_使用說明.md` — 在 ChatGPT（自訂指令 / 建立 GPT）設定步驟
- `prompts/gemini_使用說明.md` — 在 Gemini（Gem）設定步驟
- `prompts/範例對話.md` — 建議的提問方式與範例

## 這個助手擅長什麼

- 依「司法院裁判書查詢系統」等公開裁判進行系統性彙整與分析
- 處理常見姓名的「同名不同人」辨識問題
- 就當事人往來、租賃、授信等情境做風險側寫
- 引用正確的請求權基礎（民法條文等）與相似案號作為參考範本
- 產出結構化、可交付給執業律師參考的分析

## 重要原則（本包內建）

- **不含任何真實個人資料**：本 repo 及提示範例一律使用抽象化占位符，不放入任何
  真實姓名、身分證字號、地址、案號等敏感資訊。
- 一般性法律分析，**非正式法律意見**；具體個案請委任執業律師。
- 尊重無罪推定；區分「同理當事人處境」與「客觀往來風險判斷」。
- 使用者自行輸入之個案資料，請留意《個人資料保護法》，勿公開散布。

## 授權

本前置提示僅供合法之法律研究與分析用途。
